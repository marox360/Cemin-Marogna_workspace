ECU: compactRIO



compactRIO (National Instruments) used as a ECU

    Environment developed with LabVIEW, designed to manage:
  • Engine Control;
  • Data Acquisition (more than 100 signals);
  • Data Logging (up to 4GB);
  • Shifter and Clutch control;
  • Traction and Launch control.




    ECU consists in three main parts:
  • FPGA (Field Programmable Gate Array);
  • Real Time Software;
  • Host Interface.


Host GUI with engine managing tools. (maps, sensors characteristics)


    Several modifiable parameters:
  • gear shift timings;
  • TCS Threshold;
  • injection & ignitions parameters for each event and weather condition;
  • LCS clutch release curves;
  • rev limiter.


Exploded view of Engine Actuation Board.


View of Power Actuation Board.


    compactRIO manage only low-power signals, so drivers are necessary to pilot power devices.
    They are gathered in 2 power boxes:

  • Engine Actuation:
    • ignition;
    • injection.
  • Power Actuation:
    • fuel pump;
    • water pump;
    • fan;
    • shifter;
    • clutch.
